# Spring Boot 3 pet project


## Features
- REST API
- AOP logging
- Spring security implementation (lambda dsl)
- Spring rest docs implementation
- Spring data jpa and querydsl
- JWT authentication with RSA key pair
- Refresh token
- Exception handling


## Technologies
- Spring Boot 3.2
- Spring Web
- Spring Data JPA
- Spring Security
- Spring REST Docs
- Querydsl
- JSON Web Token (JWT)
- H2 Database
